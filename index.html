⚡ Phần 1 & 2

Cài Đặt OpenClaw

Chọn hệ điều hành của bạn rồi làm theo 4 bước.

Mở Terminal

Bấm Cmd + Space → gõ "Terminal" → bấm Enter

💡 Terminal là ứng dụng có sẵn trên mọi Mac, không cần cài gì thêm.

Chạy lệnh cài đặt

Copy toàn bộ dòng dưới đây, dán vào Terminal, rồi bấm Enter:

Terminal
curl -fsSL https://openclaw.ai/install.sh | bash

⏳ Đợi 1-2 phút. Script tự động cài Node.js + OpenClaw CLI + cấu hình cơ bản.

Chạy trình hướng dẫn

Terminal
openclaw onboard --install-daemon

Nhập API Key khi được hỏi (xem phần tiếp theo).

Kiểm tra thành công

Terminal
openclaw gateway status

Thấy "Gateway is running" → ✅ Xong! Mở trình duyệt → vào http://127.0.0.1:18789

Mở PowerShell

Bấm phải vào nút Start → chọn "Windows PowerShell"

⚠️ Chọn PowerShell, KHÔNG phải Command Prompt (CMD).

Chạy lệnh cài đặt

Copy toàn bộ dòng dưới đây, dán vào PowerShell, rồi bấm Enter:

PowerShell
iwr -useb https://openclaw.ai/install.ps1 | iex

⏳ Đợi 2-3 phút. Nếu bị hỏi "Execution Policy":

PowerShell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

Chạy trình hướng dẫn

PowerShell
openclaw onboard --install-daemon

Kiểm tra thành công

PowerShell
openclaw gateway status

Mở trình duyệt → vào http://127.0.0.1:18789

🔑 Phần 3

Tạo API Key (Bắt buộc)

OpenClaw cần API key để kết nối mô hình AI. Có 2 cách miễn phí:

💡 Khuyến nghị cho người mới. Miễn phí vĩnh viễn, ~1.000 lượt chat/ngày.

Vào Google AI Studio

Mở trình duyệt → truy cập: aistudio.google.com/apikey

Đăng nhập Gmail

Dùng tài khoản Gmail bất kỳ của bạn.

Tạo API Key

Bấm "Create API Key" → chọn project → bấm "Create"

Copy key

Key có dạng AIzaSy... — dán vào khi trình cài đặt yêu cầu.

🔵 Cho người dùng nâng cao. Truy cập nhiều model (Qwen, DeepSeek, Gemini, Claude, GPT) qua 1 key duy nhất.

Đăng ký OpenRouter

Vào openrouter.ai → đăng ký (Google login).

Tạo Key

Vào mục Keys → Create Key → Copy key (dạng sk-or-v1-xxxx...)

📁 Phần 8

Các File Quan Trọng — "Bộ Não" Của Agent

Hiểu các file này = hiểu cách điều khiển agent.

🧬 SOUL.md Quan trọng nhất

Đây là "linh hồn" — định nghĩa agent của bạn là ai.

  • Tính cách: Agent xưng hô thế nào, nói giọng gì
  • Vai trò: Trợ lý cá nhân? Marketing? Lập trình?
  • Nguyên tắc: Những quy tắc bất di bất dịch
  • Phong cách: Ngắn gọn hay chi tiết, dùng emoji không
  • Routing: Khi nào tự xử lý, khi nào chuyển agent khác
Ví dụ SOUL.md
# Trợ lý của tôi

## Tính cách
Bạn là trợ lý AI cá nhân. Xưng "em", gọi user là "anh/chị".

## Vai trò
- Trợ lý công việc hàng ngày
- Nhắc lịch, tóm tắt email

## Nguyên tắc
- Ngắn gọn, rõ ràng
- Không giả định — hỏi lại nếu chưa rõ
🛠️ SKILL.md Kỹ năng

Dạy agent làm một việc cụ thể. Mỗi skill = 1 thư mục chứa SKILL.md.

  • Gõ /tên-skill → agent đọc SKILL.md → thực thi → báo cáo
  • Dùng tools có sẵn: đọc/ghi file, chạy lệnh, duyệt web
  • Ví dụ: /story-bank → phân tích câu chuyện → tạo note Obsidian
📋 AGENTS.md Quy tắc

Quy tắc chung cho mọi agent: cách xử lý skill, format tin nhắn, quản lý bộ nhớ.

💓 HEARTBEAT.md Tự động

Danh sách kiểm tra mà agent đọc mỗi 30 phút. Agent tự kiểm tra các mục rồi nhắn bạn nếu cần.

Ví dụ HEARTBEAT.md
# Heartbeat checklist
- Có email quan trọng nào không?
- Có lịch hẹn nào trong 2 giờ tới?
- Nếu anh không nhắn gì hơn 8 giờ, hỏi thăm nhẹ.

🏪 Phần 9

ClawHub & Heartbeat — Mở Rộng Khả Năng

Cài thêm kỹ năng cho agent và thiết lập tự động hóa.

🏪 ClawHub — "App Store" cho Agent

clawhub.ai — nơi tìm và cài đặt skill. Tất cả miễn phí và công khai.

Cài ClawHub CLI
npm i -g clawhub
Lệnh Tác dụng
clawhub search "calendar" Tìm skill theo từ khóa
clawhub install tên-skill Cài skill mới
clawhub update --all Cập nhật tất cả skill
clawhub list Xem skill đã cài
clawhub sync --all Đồng bộ & publish skill

💓 Heartbeat — Agent tự "thức dậy" làm việc

Có sẵn trong OpenClaw. Mỗi 30 phút, agent tự kiểm tra HEARTBEAT.md và nhắn bạn nếu cần.

Cấu hình heartbeat trong openclaw.json
{
  "agents": {
    "defaults": {
      "heartbeat": {
        "every": "30m",
        "target": "last",
        "activeHours": {
          "start": "08:00",
          "end": "23:00"
        }
      }
    }
  }
}
Thuộc tính Ý nghĩa
every Tần suất: "30m", "1h", "0m" (tắt)
target Gửi đi đâu: "last" (kênh cuối), "none" (im lặng)
activeHours Chỉ chạy trong khung giờ (tránh spam ban đêm)
❓ Phần 5

Xử Lý Lỗi Thường Gặp

Bấm vào từng mục để xem cách khắc phục.

🍎 Mac: "Permission denied" ▼
Terminal
chmod +x install-mac.sh && bash install-mac.sh
🍎 Mac: Gatekeeper chặn chạy script ▼

Vào System Settings → Privacy & Security → cuộn xuống → bấm "Allow Anyway"

🪟 Windows: "Execution Policy" error ▼
PowerShell
powershell -ExecutionPolicy Bypass -File install-windows.ps1
🪟 Windows: Script bị kẹt / không phản hồi ▼

Khởi động lại máy tính, rồi chạy lại script từ đầu.

Cả hai: "openclaw: command not found" ▼

Đóng cửa sổ Terminal/PowerShell hiện tại → mở cái mới → thử lại.

Nếu vẫn không được:

Terminal
export PATH="$(npm config get prefix)/bin:$PATH"
Cả hai: Node.js not found hoặc version quá cũ ▼

Kiểm tra: node --version (cần v22+)

Nếu chưa có: vào nodejs.org → Download LTS → Cài đặt → Khởi động lại máy (Windows)
